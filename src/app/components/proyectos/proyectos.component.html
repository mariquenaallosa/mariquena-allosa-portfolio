Gardening Friend

Aplicación informativa relacionada a la horticultura para Android desarrollada utilizando Java, Firebase y Android Studio.

Funciona tanto como una biblioteca abarcando gran conocimiento relacionado a esta área, como así provista de recomendaciones de cómo explotar al máximo la actividad de realizar una huerta en casa, cuidados y ayudas diarias.

Amazing Cars

Plataforma web para una concesionaria realizada con Java,Spring Boot, MariaDB , thymeleaf, HTML, CSS y Bootstrap con un sistema de inicio de sesión para administradores y un completo panel de administración de vehículos.

Uno de nuestros mayores logros fue la exitosa implementación de operaciones CRUD, permitiendo a los administradores gestionar vehículos, marcas y tipos de vehículos de manera eficiente.

Perruqueria

Aplicación de escritorio realizado utilizando Java y Swing con la capacidad de registrar, en una base de datos, los datos tanto de mascotas como de dueños que se ingresaen desde una interfaz gráfica de usuario.
